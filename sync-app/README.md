Script para verificar se o processo do dropbox ou do google drive
está rodando na sessão do usuário especificado.
Caso não esteja rodando, inicia o processo como o usuário especificado
e na sessão que está logado.

Ao importar o template, criar o seguinte mapeamento de valor:

Sync App
0 -> Rodando
1 -> Reiniciado
2 -> Erro
